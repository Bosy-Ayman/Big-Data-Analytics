# Big Data Analytics (DSAI 427) - Spring 2025-2026

Welcome to the repository for the **Big Data Analytics (DSAI 427)** course. This repository contains all notes, assignments, lecture slides, and references organized logically.

---

## 📂 Repository Structure

- **`Lectures/`**: Presentation slides and official lecture materials.
- **`Notes/`**: Personal notes and markdown files.
  - **`assets/`**: Images and media used in notes.
- **`Assignments/`**: Code, scripts, and reports for course assignments and projects.
- **`References/`**: Textbooks, research papers, and reference PDFs.
- **`Question Bank/`**: Practice questions and past exams.

## 🚀 Overview
This course explores the ecosystem of Big Data including Hadoop, HDFS, MapReduce, YARN, and Apache Spark (SQL, Dataframes, ML, GraphX, and Streaming), preparing students to handle large-scale data processing architectures.
