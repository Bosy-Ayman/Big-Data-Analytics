GRAPH DESCRIPTIONS

GRAPH 1: scalability_graph.png
----------------------------------------
This is a dual-panel figure showing:
LEFT PANEL: Execution Time vs Number of Nodes
- X-axis: Number of Nodes (1, 2, 3)
- Y-axis: Time in seconds
- Blue line shows actual times: 42s → 28s → 23s
- Red dashed line shows ideal linear scaling

RIGHT PANEL: Speedup Analysis
- X-axis: Number of Nodes (1, 2, 3)
- Y-axis: Speedup factor
- Green line shows actual speedup: 1.00x → 1.50x → 1.82x
- Red dashed line shows ideal speedup: 1x → 2x → 3x

GRAPH 2: comparison_bar_chart.png
----------------------------------------
Bar chart comparing actual vs ideal execution times:
- Blue bars: Actual times (42s, 28s, 23s)
- Red bars: Ideal times (42s, 21s, 14s)
- Shows efficiency loss due to overhead
